## Prerequisities


To begin you'll need to have installed:

+ [Node.js](https://nodejs.org/en/)
+ npm (comes with Node.js)
    

+ Bower

    ```sh
        $ npm i -g bower
    ```  

## Installation

```bash
	bower i
```
